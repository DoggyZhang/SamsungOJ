题目描述

注意
1. This test supports C++ only.
2. None of the header files can be added to the User Code.
    (malloc.h cannot be added either，but malloc can be called)
3. The Main cannot be modified and is used as is even when being graded.
4. This certification Test conducts an exhausive review of code.
    Under any circustances, you should neither save nor try to directly access the original data.
5. Please check the main code for more info.

Lucy发现了一种新的复杂有机化合物材料，她需要一个在有机化合物材料数据库中搜索相似的复杂有机化合物的方法
一个复杂有机化合物材料由五个有机化合物组成，分别为第一有机化合物，第二有机化合物，第三有机化合物，第四有机化合物和第五有机化合物。
为了找到相似的复杂有机化合物材料，我们首先需要计算出每两个复杂有机化合物材料之间的相似度。
两个复杂有机化合物材料CC1 和CC2（如下表所示）之间，存在着五个相似度:
1) CC1和CC2的第一有机化合物的相似度
2) CC1和CC2的第二有机化合物的相似度
3) CC1和CC2的第三有机化合物的相似度
4) CC1和CC2的第四有机化合物的相似度
5) CC1和CC2的第五有机化合物的相似度

如果CC1，CC2的五个相似度中有至少一个值等于100，则可以认为CC1和CC2是相似的
CC1，CC2的总相似度定义为以上五个相似度之和

API说明
以下一个api是main提供的 user.cpp内可以直接调用
int calc_correlation(char str1[], char str2)
这个方法计算两个有机化合物之间的相似度
参数说明
str1：需要计算相似度的有机化合物
str2：需要计算相似度的有机化合物
返回值
两个有机化合物的相似度

以下API需要在user.cpp中实现
void init()
这个方法在每个测试用例的最开始调用一次
void addDB(INFO info)
这个方法将一个复杂有机化合物材料添加到数据库中
在数据库中，不存在两个相同的复杂有机化合物材料记录
换言之，不存在两个复杂有机化合物材料含有五个相同的有机化合物
参数说明
info：添加的复杂有机化合物材料的信息
int newCompound(INFO info)
这个方法是 首先在数据库中寻找与新发现的复杂有机化合物材料相似的复杂有机化合物材料，然后返回其中总相似度最高的复杂有机化合物材料。
调用 newCompound()后 addDB()不会再调用。
新发现的复杂有机化合物材料所包含的五种有机化合物一定存在于数据库中，
换言之，新发现的复杂有机化合物材料的第一有机化合物一定是数据库中各复杂有机化合物材料的第一有机化合物的 其中一个，
新发现的复杂有机化合物材料的第二有机化合物一定是数据库中各复杂有机化合物材料的第二有机化合物的 其中一个。（第三第四第五以此类推）
参数说明
info：新发现的复杂有机化合物材料的信息
返回值
返回与新发现的复杂有机化学物材料相似的 且 总相似度最高的 复杂有机化合物材料的总相似度

约束条件
1. init()方法会在每个测试用例开始时调用一次；
2. 每个测试用例中，addDB()方法最多调用5000次；
3. 每个测试用例中，newCompound()方法最多调用5000次；
4. 每个测试用例中，newCompound()方法调用后不会再调用addDB()方法；
5. 每一个复杂有机化合物材料都由五个有机化合物组成（分别是第一有机化合物，第二有机化合物，第三有机化合物，第四有机化合物和第五有机化合物）
6. 如果两种复杂有机化合物材料的五个相似度中有至少一个相似度等于100，则可以认为这两种复杂有机化合物材料是相似的
7. 每个有机化合物由一个长度不超过10的全小写字母组成的字符串表示
8. 有机化合物的种类不超过500种
   换言之，最多有500中第一有机化合物，最多有500中第二有机化合物；（第三四五有机化合物以此类推）
9. 保证组成新发现的复杂有机化合物材料的五种有机化合物都已知并存在于数据库中。

main.cpp
#ifndef _CRT_SECURE_NO_WARNINGS
#define _CRT_SECURE_NO_WARNINGS
#endif

#include <stdio.h>
#include <string.h>
#include <stdlib.h>

#define MAX_N 5000
#define MAX_M 5
#define MAX_R 500

typedef struct {
    char first[11];
    char second[11];
    char third[11];
    char fourth[11];
    char fifth[11];
}INFO;

////////////////////////////////////////////////////////////////////////////////
////////////////////////////////////////////////////////////////////////////////

static int Weight;

extern void init();
extern void addDB(INFO info);
extern int newCompound(INFO info);

int calc_correlation(const char str1[11], const char str2[11])
{
    int sum = 0;
    if (strcmp(str1, str2) == 0) return 100;
    else {
        for (int i = 0; i < 10; ++i) {
            if (str1[i] == 0 || str2[i] == 0) break;
            sum += ((26 - abs(str1[i] - str2[i])) * Weight);
        }
    }
    sum /= 100;
    if (sum > 99) sum = 99;
    return sum;
}

////////////////////////////////////////////////////////////////////////////////
////////////////////////////////////////////////////////////////////////////////

static int N, K, R[MAX_M];
static char String[MAX_M][MAX_R][11];
static int Compound[MAX_N * 2][MAX_M];

static int seed;
static int pseudo_rand() {
    seed = seed * 214013 + 2531011;
    return (seed >> 16) & 0x7fff;
}

static void make_compound()
{
    for (int i = 0; i < MAX_M; ++i) {
        for (int k = 0; k < R[i]; ++k) {
            int length = pseudo_rand() % 6 + 5;
            for (int m = 0; m < length; ++m) {
                String[i][k][m] = 'a' + pseudo_rand() % 26;
            }
            String[i][k][length] = 0;
        }
    }

    for (int i = 0; i < (N + K); ++i) {
        for (int k = 0; k < MAX_M; ++k) {
            Compound[i][k] = pseudo_rand() % R[k];
        }
    }
}

static int run()
{
    int score = 100;

    scanf("%d %d %d %d", &N, &K, &seed, &Weight);
    scanf("%d %d %d %d %d", &R[0], &R[1], &R[2], &R[3], &R[4]);

    make_compound();

    init();

    for (int i = 0; i < N; ++i) {
        INFO info;
        strcpy(info.first, String[0][Compound[i][0]]);
        strcpy(info.second, String[1][Compound[i][1]]);
        strcpy(info.third, String[2][Compound[i][2]]);
        strcpy(info.fourth, String[3][Compound[i][3]]);
        strcpy(info.fifth, String[4][Compound[i][4]]);
        addDB(info);
    }

    for (int i = N; i < (N + K); ++i) {
        int answer;
        scanf("%d", &answer);

        INFO info;
        strcpy(info.first, String[0][Compound[i][0]]);
        strcpy(info.second, String[1][Compound[i][1]]);
        strcpy(info.third, String[2][Compound[i][2]]);
        strcpy(info.fourth, String[3][Compound[i][3]]);
        strcpy(info.fifth, String[4][Compound[i][4]]);

        int result = newCompound(info);
        if (result != answer)
            score = 0;
    }

    return score;
}

int main()
{
    setbuf(stdout, NULL);
    //freopen("sample_input.txt", "r", stdin);

    int T;
    scanf("%d", &T);

    for (int tc = 1; tc <= T; tc++)
    {
        printf("#%d %d\n", tc, run());
    }

    return 0;
}
