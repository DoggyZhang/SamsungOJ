问题 A: ★置业顾问（考试版）

时间限制: 3 Sec  内存限制: 256 MB
提交: 45  解决: 22

题目描述

X公司想投资一个大型商场，恰逢政府要对一块99*26的区域进行招标，因为环境的原因，商场

建立到每块区域的利润都不同，而且有的区域可能与其他区域可获取的利润相关。假设你是负

责X公司的置业顾问，总经理需要你提供所有区域可能的利润的报表，目前你有一个助手，会

帮你陆续录入每个区域的计算公式，你负责计算出所有区域可获取的利润，基本的输入输出函

数已经提供在main.cpp中，请完成main.cpp中初始化(initTable)和更新报表(updateCell,存

在动态利润区域返回false，否则返回true)两个功能


初始化：每个区域初始（即无计算公式的区域）的利润为0
更新区域：更新(x,y)处的计算公式并更新报表


约束条件：
1、当区域形成环时，这几个区域的利润是动态的，他们最终的利润等于动态利润区域的总数


2、一个区域的利润计算公式与动态利润区域相关时，无论公式中动态利润区域是否能被消去

，都算作该区域为动态利润区域


3、每个计算公式长度不超过200个字符，共有99行和26列，行用[1-99]输入用[0-98]表示，列

用[A-Z]输入用[0-25]表示,防止数字过大，每次计算的结果用100000007取余


initTable()

void initTable()

- A function that is called whenever a test case starts and which initializes the table.

Parametes:

none

updateCell()

bool updateCell(int row, int col, char equation[], int value[][])

- Modifies the value or expression of a certain cell, renews the values of all cells in the table and returns

- In case row=0, col=0, it refers to A1 cell. In case row=98, col=25, it refers to Z99 cell

Parameters:

- row            [in] row index of modifying cell

- col             [in] column index of modifying cell

- equation     [in] expression or value of modifying cell

- value         [out] renewed result value of all cells in the table.

                   The value of each cell is divided by 100000007 and the remainder is stored

Returns:

- true           All cells being properly calculated

- false          One or more undermined cells exist （有环存在）


[ Main.cpp ] 

#include <stdio.h>

#define WIDTH	26
#define HEIGHT	99
#define LENGTH	200
const int THRESHOLD = 100000007;
static int value[HEIGHT][WIDTH];

extern void initTable();
extern bool updateCell(int row, int col, char equation[LENGTH], int value[HEIGHT][WIDTH]);

int main_init(){
	for(int i = 0; i < HEIGHT; ++i){
		for(int j = 0; j < WIDTH; ++j){
			value[i][j] = 0;
		}
	}
	int cmd = 0;
	scanf("%d", &cmd);
	return cmd;
}

int calcChecksum(int value[HEIGHT][WIDTH], bool ret){
	int sum = 0;
	for(int i = 0; i < HEIGHT; ++i){
		for(int j = 0; j < WIDTH; ++j){			
			sum += value[i][j];
			sum %= THRESHOLD;
		}
	}
	if(!ret) ++sum;
	sum %= THRESHOLD;
	if(sum < 0) sum += THRESHOLD;
	return sum;
}

int main(){
	setbuf(stdout, NULL);
	//freopen("sample_input.txt", "r", stdin);
	
	int T;
    int totalScore = 0;
	scanf("%d", &T);
	for(int tc = 1; tc <= T; ++tc){
		initTable();
		int row, col;
		char input[LENGTH];
		int checksumIn = 0;
		int cmd = main_init();        
		int score = 0;
		
		for(int i = 0; i < cmd; ++i){
			scanf("%d %d %s %d", &row, &col, input, &checksumIn);		
			if(checksumIn < 0) checksumIn += THRESHOLD;
			bool ret = updateCell(row, col, input, value);
			int checksum = calcChecksum(value, ret);			
			if(checksumIn == checksum) ++score;
		}	
		printf("#%d %d\n", tc, score);
		totalScore += score;
	}
	printf("TotalScore = %d\n", totalScore);

	return 0;
}
