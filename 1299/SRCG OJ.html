问题 A: 听歌识曲

时间限制: 10 Sec  内存限制: 256 MB

[提交][状态][讨论版]

题目描述

实现一个程序根据听到的一段歌曲识别出它的名字.

 

有两个函数需要去实现:

  1. 保存歌曲的数字信号数据

    有最多10000首歌的数据，每一首歌的长度大于等于8且小于等于200，每一个数字信号范围是-16384到16383。

  2. 在所有歌曲数据都保存之后，分析给出的一段信号来自哪一首歌

    信号长度固定为8，信号有噪声影响，所以与原始数据相比会有-128到127的误差

 

以下函数名称基于C/C++。对应的Java版本请参考Solution.java和UserSolution.java。

在Java中，char有2 byte长但是只使用低位。

 

void init(int total_music_cnt)

这个函数在每个case的开始被调用。

参数：

         Total_music_cnt ：歌曲的数量 N (5 <= N <= 10000)

void send_music_db_data(int music_id, int music_len, int data[])

这个函数传递要注册的歌曲数据。

对每一个case，music_id从1开始连续增加。

参数：

         music_id ：歌曲的id (1 <= music_id <= 10000)

         music_len ：歌曲的长度 (8 <= music_len <= 200)

         data[] ：歌曲的数字信号 (-16384 <= data[i] <= 16383)

void find_music_id(int data[])

这个函数传递要查找的歌曲中的一段。

每一个数据与原始数据相比都有-128到127的噪声。

返回查找到的歌曲的id。

data[]的长度固定为8。

参数：

         data[] ：要查找的数字信号

返回：

         与该数字信号对应的歌曲id

[限制条件]

  1. 歌曲的总数N小于等于10000.

  2. 每首歌曲的长度小于等于200.

  3. 原始的歌曲数字信号范围是-16384到+16383.

  4. find_music_id()中的噪声范围是-128到+127.

  5. 每一个case调用find_music_id()少于10000次.

  6. find_music_id()中的歌曲片段长度固定为8.

  7. 测试数据确保find_music_id()传递的数据能找到对应的原始歌曲.

  8. 测试数据确保能找到的歌曲是唯一的.

#ifndef _CRT_SECURE_NO_WARNINGS
#define _CRT_SECURE_NO_WARNINGS
#endif

#include<stdio.h>

#define MAX_MUSIC_LEN (200)
#define SAMPLE_DATA_LEN (8)

static int seed, n, max_len, query_cnt;
static int music_db[10005][MAX_MUSIC_LEN];
static int music_len[10005];

static int pseudo_rand(void);
static void make_music_db(void);
static bool make_sample_with_noise(void);

extern void init(int total_music_cnt);
extern void send_music_db_data(int music_id, int music_len, int data[MAX_MUSIC_LEN]);
extern int find_music_id(int data[SAMPLE_DATA_LEN]);

int main()
{
	setbuf(stdout, NULL);
	int tc, total_score = 0;

//	freopen("sample_input.txt", "r", stdin);

	scanf("%d", &tc);

	for (int t = 1; t <= tc; t++)
	{
		scanf("%d%d%d%d", &seed, &n, &max_len, &query_cnt);

		int correct_ans_cnt = 0;

		init(n);
		make_music_db();

		for (int query = 0; query < query_cnt; query++)
		{
			if (make_sample_with_noise())
				correct_ans_cnt++;
		}
		if (correct_ans_cnt == query_cnt)
		{
			printf("#%d 100\n", t);
			total_score += 100;
		}
		else
		{
			printf("#%d 0\n", t);
		}
	}
	printf("Total Score : %d\n", total_score / tc);

	return 0;
}

static int pseudo_rand(void)
{
	seed = seed * 214013 + 2531011;
	return (seed >> 16) & 0x7FFF;
}

static void make_music_db()
{
	int param[MAX_MUSIC_LEN];
	int music_len_param;
	for (int id = 1; id <= n; id++)
	{
		music_len_param = music_len[id] = pseudo_rand() % max_len + 1;

		if (music_len[id] < SAMPLE_DATA_LEN)
			music_len_param = music_len[id] = SAMPLE_DATA_LEN;

		for (int i = 0; i < music_len[id]; i++)
		{
			param[i] = music_db[id][i] = pseudo_rand() - 16384;
		}

		send_music_db_data(id, music_len_param, param);
	}
}

static bool make_sample_with_noise()
{
	int param[SAMPLE_DATA_LEN];

	int music_id = pseudo_rand() % n + 1;
	int offset = pseudo_rand() % music_len[music_id];

	if (offset + SAMPLE_DATA_LEN >= music_len[music_id])
	{
		offset = music_len[music_id] - SAMPLE_DATA_LEN;
	}

	for (int i = 0; i < SAMPLE_DATA_LEN; i++)
	{
		param[i] = music_db[music_id][i + offset] + pseudo_rand() % 256 - 128;
	}


	int return_val = find_music_id(param);

	return return_val == music_id;
}

class Solution {

	private static int pseudo_rand()
