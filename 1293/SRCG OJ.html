1293: ★ 日程表

时间限制: 10 Sec  内存限制: 256 MB
提交: 16  解决: 14
[提交][状态][讨论版]

题目描述

请实现一个日程表管理的程序

需要实现的API 有三个: Add Schedule, Delete Schedule, Search Schedule.

"Add Schedule"功能带有重复属性

重复属性包含类型和重复频率, 具体类型定义详见下表

类型
 描述
NONE
 不重复
DAILY
 每一日添加日程
WEEKLY
 每周添加日程
MONTHLY
 每月添加日程
YEARLY
 每年添加日程
EVERY3DAYS
 每三日添加日程
EVERY5DAYS
 每五日添加日程
EVERY10DAYS
 每十日添加日程


"Delete Schedule" 功能删除定义日期内的所有日程

根据不同的删除模式, 也可以删除所有相同id的日程


"Search Schedule"功能搜索特定时间段的日程


API说明详见下表

void init()

每个测试用例开始会被调用一次

void addSchedule(int id, DATE date, REPEAT repeat)

在日期"date"里添加日程"id"

当重复类型不是NONE,根据类型重复添加日程(具体详见下文的[例子])

id值是唯一的不会重复出现的

当重复类型是NONE时, 重复次数就是 1 , 否则重复次数值会大于等于 2 小于等于 100

添加日程的日期不会超过2099年12月31日

但重复类型是MONTHLY时, date的值 小于等于 28日

当重复类型为YEARLY时, date的值不会是2月29日

参数范围

1 <= id <= 9999

2000 <= date.year <= 2099

1 <= date.month <= 12

1 <= date.day <= 31

NONE <= repeat.type <= EVERY10DAYS

1 <= repeat.cnt <= 100

参数说明

id 添加日程的id

date 添加日程的日期

repeat 添加日程的重复属性

void deleteSchedule(DATE date, int mode)

删除定义日期内的所有日程

当mode 为 1 时,  删除所有 与被删日程具有相同id的日程

date的范围跟 addSchedule 一样


参数说明

date 打算删除日程的日期


mode 删除模式

当 mode 为 0 时, 只删除该日期中所有添加的日程

当 mode 为 1 时, 除了删除该日期的所有添加的日程, 与被删日程id相同的日程也被删除

int searchSchedule(DATE from, DATE to)

返回从 from 到 to 添加的日程的数量

to 一定大于等于 from

from 与 to 的范围跟 addSchedule 一样

搜索日期区间最大为 30日

参数说明

from 搜索的开始日期

to 搜索的结束日期

返回值

从from 到 to 添加的日程的数量


以下API在main中已经实现,user code中可以直接调用

int daysOfMonth(int yy, int mm)

返回yy年mm月的日数

返回值

yy年mm月的日数

[约束条件]

1. init()会在每个测试用开始调用一次

2. 每个测试用例,所有日期范围从2000.01.01到 2099.12.31,不会有非法输入

3. 2000.01.01是星期六

4. 添加日程时,当重复类型为NONE是, 重复次数必然为1, 否则,重复次数必然为 大于等于2小于等于100

5. 日程不会添加到2099.12.31之后

6. 当删除模式为 1 时, 所有在该日期下日程被删除,且与被删日程相同id的日程都会被删除

7. 每个测试用例,总的API调用次数, 除了init(), 小于等于10000次

typedef enum {

typedef enum {

	DAILY,
	WEEKLY,
	MONTHLY,
	YEARLY,
	EVERY3DAYS,
	EVERY5DAYS,
	EVERY10DAYS,
	TYPE_MAX
} REPEAT_TYPE;

typedef struct {
	int year;
	int month;
	int day;
} DATE;

typedef struct {
	REPEAT_TYPE type;
	int cnt;
} REPEAT;

extern void init();
extern void addSchedule(int id, DATE date, REPEAT repeat);
extern void deleteSchedule(DATE date, int mode);
extern int searchSchedule(DATE from, DATE to);

int daysOfMonth(int yy, int mm)
{
	int days;
	if (mm == 2) {
		if ((yy % 400 == 0) || ((yy % 4 == 0) && (yy % 100 != 0)))
			days = 29;
		else
			days = 28;
	}
	else if (mm == 4 || mm == 6 || mm == 9 || mm == 11)
		days = 30;
	else
		days = 31;
	return days;
}

static int run()
{
	int N, answer;
	int cmd, param1, param2, param3;
	int id = 1;

	DATE date, date2;
	REPEAT repeat;
	
	init();

	scanf("%d", &N);
	answer = 100;
	for (int i = 0; i < N; ++i) {		
		scanf("%d", &cmd);

		switch (cmd) {
		case ADD:
			scanf("%d %d %d", &param1, &param2, &param3);
			date.year = 2000 + param1 / 10000;
			date.month = (param1 / 100) % 100;
			date.day = param1 % 100;
			repeat.type = (REPEAT_TYPE)param2;
			repeat.cnt = param3;

			addSchedule(id, date, repeat);
			id++;
			break;

		case DELETE:
			scanf("%d %d", &param1, &param2);
			date.year = 2000 + param1 / 10000;
			date.month = (param1 / 100) % 100;
			date.day = param1 % 100;

			deleteSchedule(date, param2);
			break;

		case SEARCH:
			scanf("%d %d %d", &param1, &param2, &param3);
			date.year = 2000 + param1 / 10000;
			date.month = (param1 / 100) % 100;
			date.day = param1 % 100;
			date2.year = 2000 + param2 / 10000;
			date2.month = (param2 / 100) % 100;
			date2.day = param2 % 100;

			int result = searchSchedule(date, date2);
			if (result != param3) {
				answer = 0;
			}
		}
	}
	
	return answer;
}
