1291: 兴趣小组

时间限制: 10 Sec  内存限制: 256 MB
提交: 103  解决: 18

题目描述

[Description]

现在需要将一个拥有N位成员的兴趣小组按照不同的兴趣划分为两个小组。

在管理小组成员的系统中，每位成员的兴趣值是以一个整数进行存储的。

这个系统只能存储N位成员的信息，并且只有添加和删除成员的功能。

换句话说，要想改变一位成员的信息，则必须先将当前改成员的信息删除，并且被删除的信息不可被再次添加。

通过使用小组成员的信息，我们尝试将他们分为两组。

初始给定的成员信息的分布如图一所示：

现目标为将小组调整为如图二所示的分布，即两组之间有明显的区分，但不会存在太多具有极端的兴趣值的成员。

(水平有限，这图意会就好)

在当前的系统中，你可以通过反复删除图中标记的位置的成员并且添加新成员来实现前述调整。

删除具有最低或最高的兴趣值的成员，或者删除处在正中间位置的成员。

如果所有的成员是按顺序排列的，那么他们的兴趣值应该是递增的，如果多位成员的兴趣值相同，那么他们的ID值应该是递增的。

在删除几位成员的信息后，同样数量的成员信息将会被添加，故系统中将总是存储着N位成员的信息。

 

请实现管理系统具有的以下功能： 

 

[APIs]

现请根据要求实现以下API：

void init(int N, Participant mParticipants[])

初始化函数，该函数只在每一个testcase开始时调用一次。

 

Parameters

N：为兴趣小组的成员数量

mParticipants[]：每位成员的信息（0<=id,tendency<=99,999）

void addParticipant(Participant mParticipant)

添加一个成员的信息

 

Parameters

mParticipant：成员的信息（0<=id,tendency<=99,999）

int removeParticipants(int K)

删除一位成员的信息，并返回这位成员的id。

但是，如果K的值为1并且成员的数量为偶数，则删除中间的两位成员并返回他们的id之和。

例如，当一共有6位成员的信息，删除在排列后处在第3及第4位的成员的信息。

 

Parameters

K==0:删除具有最低的兴趣值的成员的信息。如果多位成员具有相同的兴趣值，删除id最小的成员的信息。

K==1:删除排序后处在中间位置的成员的信息。

K==2:删除具有最高的兴趣值的成员的信息。如果多位成员具有相同的兴趣值，删除id最大的成员的信息。

Return

被删除的成员的id。如果有两位成员被删除，返回他们的id之和。

void getTotal(int tot[])

计算兴趣值较小的一半成员的兴趣值总和以及兴趣值较大的一半成员的兴趣值总和。

若N为奇数，忽略处在中间位置的成员。

 

Parameters

tot[]:第0位存储兴趣值较低的组的总和，第1位存储兴趣值较高的组的总和。

[Constrains]

1.兴趣小组成员的数量5≤N≤50,000;

2.每个成员的兴趣值0≤tendency≤99,999;

3. 如果所有的成员是按顺序排列的，那么他们的兴趣值应该是递增的，如果多位成员的兴趣值相同，那么他们的ID值应该是递增的。

4.当调用removeParticipants()并且K = 1 并且 N为偶数时，删除两位成员信息；

5.当一位成员信息被删除时，一位新成员的信息将会立刻被添加，并且系统中将始终存储着N位成员的信息；

6.当getTotal()被调用时，返回的结果保证小于2,000,000,000；

7.调用API的总次数小于15,000次；


C++ Main: 

#ifndef _CRT_SECURE_NO_WARNINGS 
#define _CRT_SECURE_NO_WARNINGS 
#endif

#include<stdio.h>

#define MAXID 100000
#define MAXTENDENCY 100001

#define REMOVERATE 6
#define AVERAGERATE 4

typedef struct {
	int id;
	int tendency;
} Participant;

extern void init(int N, Participant mParticipants[]);
extern void addParticipant(Participant mParticipant);
extern int removeParticipants(int K);
extern void getTotal(int tot[]);

static int seed = 3;
static int N, M;

static int idChk[MAXID];
static int accepted;

static bool conChk;

static int pseudo_rand(void)
{
	seed = seed * 431345 + 2531999;
	return seed & 0x7FFFFFFF;
}

static int getTendency() {
	int tendency;

	do {
		tendency = pseudo_rand() % MAXTENDENCY;
	} while (tendency == MAXTENDENCY - 1);

	return tendency;
}

static void initFunc(int test_case, int n) {

	Participant participants[50001];

	for (int i = 0; i < n; ++i) {
		int id = pseudo_rand() % MAXID;
		int tendency = getTendency();

		while (idChk[id] == test_case) {
			id = (id + 1) % MAXID;
		}

		idChk[id] = test_case;

		participants[i].id = id;
		participants[i].tendency = tendency;
	}

	init(n, participants);
}

static void addFunc(int test_case) {

	Participant participant;

	int id = pseudo_rand() % MAXID;
	int tendency = getTendency();

	while (idChk[id] == test_case) {
		id = (id + 1) % MAXID;
	}

	idChk[id] = test_case;

	participant.id = id;
	participant.tendency = tendency;

	addParticipant(participant);
}

static int run(int test_case) {
	
	accepted = 0;

	bool conChk = true;

	initFunc(test_case, N);

	while (M--) {
		int command = pseudo_rand() % (REMOVERATE + AVERAGERATE);

		if (conChk && command < AVERAGERATE) {
			int tot[2];
			int correctTot[2];

			getTotal(tot);

			scanf("%d%d", &correctTot[0], &correctTot[1]);

			if (tot[0] == correctTot[0] && tot[1] == correctTot[1]) {
				++accepted;
			}

			conChk = false;
		}
		else {
			int K = pseudo_rand() % 3;
			int retId, correctId;

			retId = removeParticipants(K);

			scanf("%d", &correctId);

			if (retId == correctId) {
				++accepted;
			}

			addFunc(test_case);
			if (K == 1 && N % 2 == 0) {
				addFunc(test_case);
			}

			conChk = true;
		}
	}

	return accepted;
}

int main(void) {
	int T;
	int correct, totalScore = 0;

	// freopen("sample_input.txt", "r", stdin);

	scanf("%d", &T);

	setbuf(stdout, NULL);

	for (int test_case = 1; test_case <= T; ++test_case) {

		scanf("%d%d%d%d", &seed, &N, &M, &correct);

		if (run(test_case) == correct) {
			printf("#%d 100\n", test_case);
			totalScore += 100;
		}
		else {
			printf("#%d 0\n", test_case);
		}
	}

	printf("Total Score = %d\n", totalScore / T);

	return 0;
}
